## **The System for Submitting and Approving Conduct Grades using Spring Microservices Ecosystem.**

## Techstack for the project:

### Backend:
- Java Spring Boot
- Spring Cloud
- OpenFeign
- API Gateway
- Eureka
- Spring Data JPA
- Spring Security
- JWT
- Kafka
- Websocket

### Frontend:
- ReactJS
- Redux
- Ant Design
- Websocket

### Other tools:
- Docker, Docker Compose for building the app
